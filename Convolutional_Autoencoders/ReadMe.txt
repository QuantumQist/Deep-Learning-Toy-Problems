This folder contains an analysis of MNIST dataset with convolutional autoencoders.

The main notebook is
* `Analysis_of_MNIST_dataset_with_ConvNets.ipynb` <- here you will find the main results

The supporting files are
* `engine.py` contains the training functionalities
* `model_classes.py` contains the classes with neural network architectures
* `utils.py` contains the utility functions
